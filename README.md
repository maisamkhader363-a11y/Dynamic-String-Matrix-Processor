# Dynamic String Matrix Processor (C++)

An efficient C++ implementation demonstrating advanced memory management using 2D dynamic arrays (`char**`) to store and process user-defined strings.

## 🚀 Features
- **Dynamic Allocation:** Dynamically allocates memory for both rows and columns based on user input.
- **Vowel Counting (Individual):** Counts and displays vowels for each string separately.
- **Vowel Counting (Global):** Calculates the total number of vowels across the entire matrix.
- **Longest String Finder:** Iterates through the matrix to locate the longest string efficiently.
- **Memory Management:** Ensures proper cleanup using `delete[]` to prevent memory leaks.

## 🛠️ Concepts Covered
- Pointers and Multi-dimensional Dynamic Arrays (`char**`)
- String Manipulation (`strlen`, Character checking)
- Modular Programming (Functions & Switch-case menus)
- Safe Memory Deallocation
